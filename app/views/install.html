JPEG2000 Conversion GUI

Installation

Before using this tool you must have executables installed that can convert an image into a JPEG2000 (JP2) image. You have two choices. You can either install the open source OpenJPEG tools or the proprietary Kakadu ones.

Install OpenJPEG to have the opj_compress and opj_decompress commands in your PATH.

Install the Kakadu executables and have the kdu_compress and kdu_expand commands in your PATH. On a Mac or Linux system you can check that these are installed by typing the following into a console: which kdu_compress. If you see the path to the command then it is installed and you are ready to go!
