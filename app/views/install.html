JPEG2000 Conversion GUI

Before using this tool you must install the Kakadu executables and have the kdu_compress and kdu_expand commands in your PATH. On a Mac or Linux system you can check that these are installed by typing the following into a console: which kdu_compress. If you see the path to the command then it is installed and you are ready to go!
